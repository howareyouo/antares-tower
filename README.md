## 快速开始

+ node版本：v10.4.1；

+ 安装：

	```
	cnpm install
	```

+ 运行项目：

	```
	cnpm build
	cnpm start
	```

+ 访问项目：[localhost:3000](http://localhost:3000) 

+ 构建项目：

	```
	./deploy.sh
	```
